# OpenGL Battleship Game

A fully interactive, classic Battleship strategy game implemented entirely in C++ using the OpenGL graphic library and GLUT. It features custom low-level rendering pipelines, mouse/keyboard event handling, and complex grid-based game logic.

## 🚀 Key Features

* **Custom Graphics Pipeline**: Renders a dynamic ocean grid, interactive ship placements, and hit/miss animations using pure OpenGL primitives and GLUT.
* **Complex State Management**: Handles complex game states including ship rotation, placement collision detection, turn-based logic, and win/loss condition tracking.
* **Interactive UI**: Fully functional mouse tracking for grid-based interactions, coordinate mapping, and keyboard shortcuts for game controls.

## 🛠️ Technology Stack

* **Language**: C++
* **Graphics API**: OpenGL
* **Window Utility**: GLUT (FreeGLUT / GLEW)
* **Image Processing**: CImg (for custom texture loading)
